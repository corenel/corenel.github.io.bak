前言

在Twitter、Facebook、Google+、Weibo等社交网站，以及Evernote、Onenote等笔记应用与如此丰富与成功的今天，为什么要建Blog这么一个看起来快要被时代淘汰的东西？我想，Blog相比于其他的社交媒体或是笔记应用至少有以下好处：

  • Blog更有条理。现在的轻媒体注重的是时效性，基本上都是以Timeline为主线，碎片化严重。而Blog则是以Page为主，可以通过Category以及Tag来组织，更有条理，更容易利于以后的自我审阅与回顾。
  • Blog更易专注。社交媒体太乱，Blog则是自己的，只有自己来慢慢写，能把自己的想法完全地展现出来。
  • Blog更为公开。Evernote、Onenote等虽然也有分享功能，但是太弱了，基本上只能单个分享，各个笔记本之间没有联系，不便于网上的朋友浏览。而Blog则没有这种不足。

当然你也可以看看这篇文章。

而使用Github Pages与Hexo来建站，则更有学习使用Github、能用Markdown语言写Blog等优点，以下不再赘述。

什么是Github Pages？

Github Pages是Github推出的一个服务，是用来为你托管在Github上的项目作介绍的网站。
Github Pages事实上也是Github上的一个Repository，不过Github专门为它给了子域名（username.github.io，username是你的用户名），所以你可以放静态网页上去，用来做博客。

什么是Hexo

Hexo是一个简单、快速、强大的博客发布工具，支持Markdown格式。

购买域名

推荐上Godaddy购买，安全便捷。虽然是个国外网站，但是出人意料的支持支付宝支付。

  1. 查询你想要的域名
  2. 如果该域名Available，点击「Continue to Cart」继续
  3. 无视Godaddy的收费服务，继续「Continue to Cart」

  4. 此处可以修改购买年限，默认是两年。除了域名的购买费用之外，还有ICANN的管理费用。

    网上可以搜到Godaddy的各种优惠码，填在下面即可。截至2015.02.02，我只找到过第一年优惠为9.92美元的，据说之前有优惠到8.99美元的，看各位自己的运气了。
    有些优惠码只支持海外的信用卡支付，而不支持支付宝支付，各位需要甄别。
    若遇到输入优惠码却没有优惠的情况，请清楚浏览器Cookies再试。

  5. 最后就是账单的结算，填完必要信息后可以选择使用支付宝结算。支付完成后记得在邮箱内查看激活邮件进行激活。
