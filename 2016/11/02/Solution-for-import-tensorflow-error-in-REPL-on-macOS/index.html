Posted on 2016-11-02 | Post modified 2016-11-02 | In Experience |

Description

Solution

经检查是protobuf的锅. TF 需要protobuf>=3.0.0a3, 而macOS里似乎有两份protobuf, 一份是之前装的2.6.1, 另外一份是随着 TF 装的. 默认似乎是调用到了 2.6.1 的那个版本.
